# Telegram support bot (aiogram) — webhook, для Render

Коротко: бот приймає питання від користувачів (дві теми, українською), пересилає адміну(ам) з інформацією про автора, адмін відповідає через команду `/ans ID текст`.

1) Кроки:
- Відкрийте репозиторій на GitHub і додайте файли: `requirements.txt`, `main.py`, `render.yaml` (опціонально), `.gitignore`.
- Підключіть репозиторій до Render → New → Web Service.
- В Render у параметрах сервісу вкажіть:
  - Build Command: `pip install -r requirements.txt`
  - Start Command: `python main.py`
- Додайте Environment Variables:
  - `BOT_TOKEN` — токен від @BotFather (якщо токен був опублікований, перегенеруйте токен у BotFather).
  - `ADMIN_IDS` — ваш Telegram ID (можна кілька через кому). Наприклад: `7485071184`
  - `WEBHOOK_URL` — https://<your-service>.onrender.com  (после создания сервиса Render даёт URL — вставьте его и перезапустите)
- Деплойте. В логах має з'явитись повідомлення про встановлення webhook.

2) Тестування:
- Відкрийте чат з ботом, натисніть /start.
- З іншого аккаунта надішліть питання через кнопку.
- В адміна в чаті з ботом з'явиться повідомлення з кнопкою "Відповісти".
- Натисніть кнопку → бот підкаже формат → відправте: `/ans <user_id> Текст відповіді`.
- Користувач отримає повідомлення від бота.

3) Примітка з безпеки:
Якщо токен було оприлюднено раніше — перегенеруйте токен у @BotFather і замініть BOT_TOKEN у Render.
